gpiomockup package¶

Module contents¶

Contains a RPi GPIO library test mockup.

Currently the official GPIO is not fully implemented. Never the less this module is useful for the most common scenarios.

set_mapfile(mapfile, initialize: bool = False)¶

Set a mapfile for possibly sharing between processes.

Parameter:
  • mapfile – This can be either a path to a writable physical file or a fileno of an already opened file (mode ‚r+b‘)
  • initialize (bool) – Set to True for cleaning up the file initialize it’s content. Initialize means to set the directions to 255 (neither IN nor OUT), set all PIN values to LOW and all resistors to PUD_OFF.
