config module¶

Alle hier verwendeten PINs beziehen sich auf den BOARD-Mode, also die RPi-Nummerierung am Header.

config.BOARDFILE = 'board.json'¶

Name der Datei in der der Boardstatus

config.BTN_DURATION_REBOOT = 2.0¶

Dauer in Sekunden, die der Shutdown-Button gedrückt sein muss, um einen Reboot auszulösen. Achtung: dieser Wert muss KLEINER als BTN_DURATION_SHUTDOWN sein

config.BTN_DURATION_SHUTDOWN = 6.0¶

Dauer in Sekunden, die der Shutdown-Button gedrückt sein muss, um einen Shutdown auszulösen. Achtung: dieser Wert muss GRÖSSER als BTN_DURATION_REBOOT sein

config.CAM_FRAMERATE = 10¶

Bildrate

config.CAM_HEIGHT = 480¶

Höhe des gestreamten Kamerabildes

config.CAM_PORT = 8000¶

Port des Kameraservers

config.CAM_WIDTH = 640¶

Breite des gestreamten Kamerabildes

config.CONTROLLER_HOST = 'localhost'¶

Host des XMLRPC-Controller-Server

config.CONTROLLER_PORT = 8010¶

Port des XMLRPC-Controller-Server

config.CONTROLLER_URI = 'http://localhost:8010'¶

Adresse des XMLRPC-Server, der die Board-Schnittstelle bereitstellt

config.DAWN_OFFSET = -1800¶

Anzahl Sekunden nach Sonnenaufgang, in der die Tür geöffnet wird

config.DOORCHECK_INTERVAL = 60¶

Prüfintervall des Türstatus (Öffnen / Schließen) in Sekunden

config.DOOR_AUTOMATIC_OFFTIME = 1800¶

Dauer in Sekunden die die Türautomatik bei manueller Bedienung deaktiviert wird.

config.DOOR_AUTO_DEACTIVATED = -1¶

Türautomatik dauerhaft deaktiviert

config.DOOR_AUTO_OFF = 0¶

Türautomatik vorübergehend deaktiviert

config.DOOR_AUTO_ON = 1¶

Türautomatik aktiv

config.DOOR_CLOSED = 8¶

Tür ist geschlossen

config.DOOR_MOVE_DOWN_TIME = 6.6¶

Maximale Zeit zum Schließen der Tür (Sekunden)

config.DOOR_MOVE_UP_TIME = 8.5¶

Maximale Zeit die die Tür zum Öffnen benötigt

config.DOOR_MOVING = 3¶

Tür in Bewegung

config.DOOR_MOVING_DOWN = 2¶

Bewegung nach unten (Tür schließt sich)

config.DOOR_MOVING_UP = 1¶

Bewegung nach oben (Tür öffnet sich)

config.DOOR_NOT_MOVING = 0¶

Initialstatus, eigentlich „unknown“

config.DOOR_OPEN = 4¶

Tür ist offen

config.DUSK_OFFSET = 2400¶

Anzahl Sekunden nach Sonnenuntergang, in der die Tür geschlossen wird.

config.EARLIEST_OPEN_TIMES = {0: (7, 0), 1: (7, 0), 2: (7, 0), 3: (7, 0), 4: (7, 0), 5: (8, 0), 6: (8, 0)}¶

Zeitpunkt des frühesten Öffnens der Tür

config.INPUT_PINS = (13, 11, 29)¶

Menge aller Eingabe-Pins

config.LATITUDE = 51.138904¶

Breite

config.LIGHT_INDOOR = 36¶

PIN für das Innenlicht-Relais (230V)

config.LIGHT_OUTDOOR = 31¶

PIN für das Aussenlicht-Relais (230V)

config.LOGDATEFMT = '%m-%d %H:%M:%S'¶

Template für das Zeitformat der Logausgabe (siehe logging - Modul)

config.LOGDIR = 'log'¶

Log-Verzeichnis relativ zu www (root)

config.LOGFORMAT = '%(asctime)s %(name)-20s %(levelname)-8s %(message)s'¶

Template für die Logausgabe (siehe logging - Modul)

config.LOGLEVEL = 20¶

Logging Level, 10 = logging.DEBUG

config.LONGITUDE = 12.094316¶

Länge

config.LOWER_REED_OFFSET = 0.6¶

Dauer in Sekunden die die Tür nach Signalisierung

config.MAPFILE = -1¶

Auf -1 setzen wenn Release!

config.MAX_STREAM_COUNT = 3¶

Maximale Anzahl paralleler Streams

config.MAX_STREAM_TIME = 300¶

Maximale Zeit, die ein Stream offengehalten wird (in Sekunden)

config.MOTOR_ON = 40¶

PIN für das Relais mit der Motor-An/Aus-Schaltung (19V)

config.MOVE_DIR = 38¶

PIN für das Relais mit der Drehrichtung (siehe MOVE_UP, MOVE_DOWN)

config.MOVE_DOWN = 0¶

Schaltwert für das Relais MOVE_DIR nach unten

config.MOVE_UP = 1¶

Schaltwert für das Relais MOVE_DIR nach oben

config.OUTPUT_PINS = (40, 38, 36, 31)¶

Menge aller Ausgabe-Pins

config.PIN_BUTTON_1 = 29¶

PIN für den Button 1 (Pullup, HIGH = offen / nicht gedrückt)

config.PIN_REED_1 = 13¶

PIN für den Magnetschalter 1 (Pullup, HIGH = offen / kein Kontakt)

config.PIN_REED_2 = 11¶

PIN für den Magnetschalter 2 (Pullup, HIGH = offen / kein Kontakt)

config.PIN_RELAIS_1 = 40¶

PIN für Relais 1

config.PIN_RELAIS_2 = 38¶

PIN für Relais 2

config.PIN_RELAIS_3 = 36¶

PIN für Relais 3

config.PIN_RELAIS_4 = 31¶

PIN für Relais 4

config.RAISE_ERRORS = False¶

Gibt an, ob zusätzlich zur Fehlerausgabe im Logger eine Exception ausgelöst werden soll.

config.REED_CLOSED = 0¶

Input für Magnetkontakt geschlossen (Kontakt)

config.REED_LOWER = 11¶

Unter Magnetkontakt

config.REED_OPENED = 1¶

Input für Magnetkontakt offen (kein Kontakt)

config.REED_UPPER = 13¶

Oberer Magnetkontakt

config.RELAIS_OFF = 1¶

Output für Relais offen (rot)

config.RELAIS_ON = 0¶

Output für Relais geschlossen (grün)

config.RELAIS_PINS = (40, 38, 36, 31)¶

Menge aller Relais-Pins.

config.RESOURCEDIR = 'resources'¶

Resourcen-Verzeichnis relativ zu www (root)

config.SCRIPTDIR = 'scripts'¶

Script-Verzeichnis relativ zu www (root)

config.SENSORFILE = 'sensor.csv'¶

Sensorwerte, relativ zu RESOURCEDIR

config.SENSOR_INTERVALL = 300¶

Intervall in dem die Messergebnisse der Sensoren erfasst werden in Sekunden.

config.SENSOR_LINE_TPL = '%d;%.2f\n'¶

Template für die Ausgabe einer Zeile im SENSORFILE. Die Substituierungswerte sind (light, temperature).

config.SHUTDOWN_BUTTON = 29¶

Shutdown-Button

config.SUNRISE_INTERVAL = 7000¶

Intervall, in dem die Sonnenaufgangs / Untergangszeiten berechnet werden (in Sekunden)

config.SWITCH_LIGHT_OFF_AFTER_CLOSING = 0¶

Wie SWITCH_LIGHT_ON_BEFORE_CLOSING, allerdings die Ausschaltzeit nach dem Schließen. Ist nur relevant, wenn SWITCH_LIGHT_ON_BEFORE_CLOSING != 0

config.SWITCH_LIGHT_ON_BEFORE_CLOSING = 0¶

Gibt an, wieviel Sekunden vor den Schließen der Tür die Innen- beleuchtung aktiviert werden soll. 0 = aus

config.TFT_SLEEP_TIMEOUT = 30¶

Anzahl Sekunden nach der der TFT ohne Aktivität (Touch) ausgeschalten wird

config.UPPER_REED_OFFSET = 0.6¶

Wie LOWER_REED_OFFSET für den oberen
