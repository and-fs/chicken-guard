class tests.base.Future(function, *args, **kwargs)¶

Bases: object

Future-Implementierung für die Tests. Führt eine Funktion nebenläufig aus, das Ergebnis kann dann zu einem späteren Zeitpunkt abgefragt werden.

Beispiel:

import time
def time_consuming_pow2(a):
    time.sleep(2.0)
    return "%d * %d is %d" % (a, a, a * a)

f = Future(time_consuming_pow2, 10)
print (f.HasResult()) # False
time.sleep(1.0)
print (f.HasResult()) # False
print (f.WaitForResult(2.0)) # "10 * 10 is 100"

WaitForResult(waittime=None)¶

Waits waittime seconds (or fractions thereof) for the contained function to be executed and returns it’s result.

If waittime has been reached without getting a result, a TimeoutError is raised.
